# 中文网站内部评审指南

本轮重点评审中文网站的内容、样式、品牌感和使用体验。

以下内容不在本轮评审范围：

- 联系表单真实邮件发送
- 服务器与部署
- 域名与 HTTPS
- 多语言功能

补充说明：

- 本轮不设置独立客户案例页面。
- 首页保留简短匿名案例介绍。

## 评审链接清单

### 首页

- `/`

建议重点看：

- 第一印象是否符合 Acoeurs Consulting 的定位？
- 标题是否清楚说明公司做什么？
- 页面长度是否合适？
- 企业服务和个人服务的主次是否清楚？
- 图片、颜色、字体和动画是否专业？
- 首页案例介绍是否应保留当前长度？

### 企业服务总览

- `/business/`

建议重点看：

- 企业服务整体结构是否清楚？
- 五类服务划分是否符合实际业务？
- 页面信息量是否合适？
- 联系入口是否自然？

### 企业服务详情页

- `/business/market-entry/`
- `/business/company-banking/`
- `/business/tax-legal-compliance/`
- `/business/local-operations/`
- `/business/growth/`

建议重点看：

- 五类服务划分是否符合实际业务？
- 服务名称是否准确？
- 页面是否过长或过短？
- 是否存在重复内容？
- CTA 是否自然？
- 每页的重点是否足够明确？

### 个人服务总览

- `/personal/`

建议重点看：

- 企业服务与个人服务的区分是否清楚？
- 页面风格是否与企业服务协调？
- 个人服务整体是否容易理解？

### 个人服务详情页

- `/personal/residency-family/`
- `/personal/property-wealth/`
- `/personal/cross-border-tax-risk/`

建议重点看：

- 三类服务划分是否完整？
- 是否存在容易让客户误解的承诺？
- 页面风格是否与企业服务协调？
- 文案是否足够克制、专业、可信？

### 关于我们

- `/about/`

建议重点看：

- 是否准确表达公司定位？
- 巴黎、上海、香港的表述是否合适？
- 是否需要增加团队、数据或照片？
- 是否存在内容不足或重复？

### 联系我们

- `/contact/`

建议重点看：

- 三步问卷是否容易理解？
- 是否能快速完成？
- 身份分类和服务分类是否准确？
- 是否还需要减少字段？
- 移动端操作是否方便？

### 法律与隐私页面

- `/legal/`
- `/privacy/`
- `/cookies/`

建议重点看：

- 页面是否容易阅读？
- 信息是否足够克制、清楚？
- 是否有明显不合适或不该公开的内容？

## 全站统一评审问题

- Header 和 Footer 是否清楚？
- 导航是否容易理解？
- “联系我们”按钮是否过多或过少？
- 深蓝、金色和米色是否符合品牌？
- 页面之间是否像同一个网站？
- 哪些区域最需要调整？
- 哪些页面最成熟，哪些页面最需要继续打磨？

## 反馈提醒

- 尽量说明具体页面和具体位置。
- 不要只写“整体不好看”。
- 请区分事实错误、业务错误和个人审美。
- 同一个问题尽量只提交一次。
- 如有参考网站或图片，可附上链接或截图。
